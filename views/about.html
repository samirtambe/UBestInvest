About

Our primary goal is to introduce investing to young people and novice investors so that they may develop an interest in finance. Secondly, we want to enhance their overall investment acumen so they may benefit from it throughout their lives. We aim to give users a place where they can act like investors with real financial data and expand their investment horizons.

APIs

We are passionate about using the latest and greatest technologies to for the benefit and convenience of our customers. We use Quandl for investment data. Business news related information is provided by The New York Times Developer API.

Technology Stack

Angular.js Lets you extend HTML vocabulary for your application. The resulting environment is extraordinarily expressive, readable, and quick to develop.
Bootstrap The most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.
CSS3 Adds styling to HTML. CSS3 extends its predecessor by adding many new features such as rounded corners, shadows, gradients, transitions or animations, as well as new layouts like multi-columns, flexible box or grid layouts.
D3.js A JavaScript library for manipulating documents based on data. D3 helps you bring data to life using HTML, SVG, and CSS.
HTML5 The core language of nearly all Web content. It is the language that describes the structure and the semantic content of a Web document.
JavaScript A lightweight, interpreted, programming language with first-class functions. Most well-known as the scripting language for Web pages, many non-browser environments also use it, such as node.js and Apache CouchDB.
Spin.js A JavaScript library that provides a visual indicator that processing or network activity is occuring.
SVG Scalable Vector Graphics is an XML-based markup language for describing two-dimensional vector graphics.
UI Router The de-facto solution to flexible routing with nested views in AngularJS.
